You are classifying one newspaper issue transcript that may contain:
- zoning law text,
- zoning hearing notices,
- zoning discussion without law text,
- non-zoning legal notices/ordinances, and
- unrelated content.

Important scope:
- Classify conservatively.
- "full code" is rare and should only be used when the text itself shows broad, standing zoning code structure as published legal text.
- If uncertain between full code vs amendment/notice/discussion, do NOT choose full code.

Primary objective:
1) Detect whether any zoning content is present.
2) If present, classify what kind of zoning material it is.
3) Distinguish comprehensive full zoning code from narrower amendments/notices.

Definitions:
- zoning anchor: explicit references such as "zoning", "zoning ordinance", "rezoning", "zoning map", zoning district changes (e.g., R-1 to C-1), zoning board/commission, variance, conditional/special use.
- zoning legal text: operative or proposed legal language that actually enacts/changing zoning rules/order.
- hearing notice: formal notice of a zoning hearing with schedule/place context.
- zoning discussion: zoning is discussed/reported but legal text is not printed.
- ordinance jurisdiction: the governing city/town/village/township/county/state named in the legal text (not necessarily the newspaper's publication city).

Full-code standard (very strict):
Use full_code_comprehensive or full_code_recodification only when ALL are true:
- The issue prints standing citywide zoning code text (not only one parcel/district action, not only hearing/notice logistics, not only article summary).
- The text includes ordinance-style legal structure (for example repeated article/section clauses or equivalent codified legal blocks), not merely narrative statements saying rules exist.
- At least 3 of these 4 components are explicitly visible in printed legal text:
  1. district framework/classifications,
  2. use rules (permitted/prohibited/conditional uses),
  3. dimensional standards (lot size, setbacks, height, density, parking),
  4. procedures/enforcement/administration (boards, permits, appeals, penalties, definitions).
- There is explicit adoption/recodification context tied to that code text.

Hard vetoes for full_code_*:
- If the issue is mainly hearing/public notice language (for example hearing date/time/place, "copy may be examined", publication notices) and does not print substantial codified zoning sections, do NOT use full_code_*.
- If the issue only summarizes zoning changes in news prose (for example "rules are set out", "ordinance adopted") without substantial codified clauses, do NOT use full_code_*.
- If zoning appears only as cross-reference inside another policy domain (for example cannabis licensing, traffic, health/sanitary, finance/bonds), use non_zoning_ordinance unless the text itself prints substantive zoning code clauses.

Class guidance:
- full_code_comprehensive: enacted in-force comprehensive code publication.
- full_code_recodification: recodifying/replacing/consolidating zoning code publication (may be proposed or operative).
- comprehensive_code_excerpt: broad code-like zoning excerpt, but insufficient adoption/recodification proof OR insufficient codified component coverage for full_code_*.
- amendment_substantial_text: amendment/revision with long replacement legal text.
- amendment_targeted_text: focused legal change (limited clauses/sections).
- map_rezoning_order: rezoning map/order for specific parcels/areas.
- variance_special_use_order: variance/special-use action language.
- procedural_notice_only: formal zoning notice/hearing/adoption notice without substantial printed zoning code clauses.
- non_zoning_ordinance: legal ordinance text but not zoning-law text.
- none: no legal-document signal.
