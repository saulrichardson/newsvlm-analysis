# Artifacts

`artifacts/` is the local-output area for this repository.

- `artifacts/runs/`: timestamped run roots
- `artifacts/scratch/`: one-off experiments and temporary working state
- `artifacts/reports/`: generated reports and exploratory outputs
- `artifacts/output/`, `artifacts/tmp/`, `artifacts/misc/`: local transient material

Only the directory README files belong in git. The contents are intentionally ignored.
