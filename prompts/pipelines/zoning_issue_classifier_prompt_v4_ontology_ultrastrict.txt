You are classifying one newspaper issue transcript that may contain:
- zoning law text,
- zoning hearing notices,
- zoning discussion without law text,
- non-zoning legal notices/ordinances, and
- unrelated content.

Important scope:
- Classify conservatively.
- "full code" is rare.
- If uncertain between full code vs amendment/notice/discussion, do NOT choose full code.

Primary objective:
1) Detect whether any zoning content is present.
2) If present, classify what kind of zoning material it is.
3) Distinguish comprehensive full zoning code from narrower amendments/notices.

Core definitions:
- zoning anchor: explicit zoning terms (zoning ordinance, rezoning, zoning map, district reclassification, variance/special use, zoning board/commission).
- zoning legal text: operative/proposed legal language that enacts or changes zoning rules.
- hearing notice: formal notice with hearing logistics (date/time/place) about zoning action.
- zoning discussion: zoning discussed in narrative/reporting, without printed legal text.
- ordinance jurisdiction: the governing place named in the legal text (city/town/village/township/county/state), which may differ from newspaper publication location.

Full-code decision (ultra-strict):
Use full_code_comprehensive or full_code_recodification only when ALL are true:
1) The issue prints substantial codified zoning ordinance language (not just summary prose, not only hearing/notice logistics, not only one narrow amendment).
2) The text shows ordinance-style structure (articles/sections/clauses) across multiple provisions.
3) ALL 4 components are explicitly visible in printed legal text:
   - district framework/classifications,
   - use rules,
   - dimensional standards (numeric/explicit standards such as lot area, setbacks, height, density, parking),
   - procedures/enforcement/administration.
4) There is explicit adoption/recodification context tied to that code text.

If ANY of the above is missing, do NOT use full_code_*.
Prefer comprehensive_code_excerpt (or amendment_* / notice classes as appropriate).

Hard vetoes for full_code_*:
- Public-notice-dominant documents (hearing logistics, publication notices, "copy may be examined") without substantial codified text -> procedural_notice_only.
- Narrative summaries saying rules exist, without printing those rules -> comprehensive_code_excerpt.
- Cross-domain ordinances (cannabis, traffic, sanitary/health, finance, partnerships, elections, etc.) that merely reference zoning or districts -> non_zoning_ordinance unless substantive zoning code clauses are the dominant legal text.
- "Continued from page" fragments or partial installments lacking clear comprehensive context should not be promoted to full_code_* unless the visible fragment itself clearly satisfies all criteria above.

Class guidance:
- full_code_comprehensive: enacted in-force comprehensive citywide zoning code text.
- full_code_recodification: recodifying/replacing/consolidating zoning code text (operative or proposed).
- comprehensive_code_excerpt: broad zoning code-like text but insufficient proof for full_code_*.
- amendment_substantial_text: amendment with long replacement legal text.
- amendment_targeted_text: focused legal changes.
- map_rezoning_order: parcel/area reclassification map order.
- variance_special_use_order: variance/special-use action order.
- procedural_notice_only: formal notice/adoption-hearing notice without substantial codified zoning text.
- non_zoning_ordinance: legal ordinance text not substantively zoning.
- none: no legal-document signal.
